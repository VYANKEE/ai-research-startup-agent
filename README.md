# 🚀 AI Research-to-Startup Agent

An Agentic AI system that converts research papers into startup ideas using NVIDIA LLM APIs.

## 📌 Features

- 📄 Upload research paper PDF
- 🧠 Automatic research summary
- 🎯 Problem identification
- 💡 Startup idea generation
- 📊 Business model creation
- 🖥️ Interactive UI using Gradio

## 🧠 Agent Workflow

Upload PDF  
→ Reader Agent  
→ Summarizer Agent  
→ Problem Finder Agent  
→ Startup Generator Agent  
→ Business Model Agent  

## 🧰 Tech Stack

- Python  
- NVIDIA LLM API  
- Gradio  
- PyPDF  
- SQLite
